Crazyswarm
===========

.. note::

   For more information, please check out the `Crazyswarm Documentation <https://crazyswarm.readthedocs.io/en/latest/>`__

Overview
^^^^^^^^^
Crazyswarm is a software stack that allows researchers to interface with real crazyflies platforms. It provides the ROS interface, handles radio communication with quadcopters, and works with motion capture system to localize quadcoptors. Additionally, it has many useful built-in functionalities, such as an integrated state-estimation pipeline using Extended Kalman Filter. All in all, it is designed for researchers to handle multi-quadcoptor/ swarm experiments.
