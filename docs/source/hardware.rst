Crazyflies
^^^^^^^^^^^

#. VICON marker placement: Each quadcopter is required to have a distinct marker placements, which enables the VICON system to distinguishes and track multiple quadcopters at once.


VICON
^^^^^^

#. Create crazyflie model in VICON: After placing VICON markers on a crazyflie, it needs to be calibrated in the VICON system computer and created an instance in the VICON software.
